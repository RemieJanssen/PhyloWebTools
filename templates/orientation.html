Create a form with a text field for a list of graph edges and a dropdown selection of a graph class (TC, SF, O, TB, all). When the user submits the form, the server will return a list of all orientations of the graph that are within this class. The list is served as a selection box. This action of the server is done by calling the function _orientation_page from app.py. The user can then select one of the orientations and the server will return the corresponding graph.
